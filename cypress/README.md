# Testes E2E - Gerador de Jogos PDG

Suite de testes end-to-end utilizando Cypress para o aplicativo Gerador de Jogos PDG.

## Estrutura

```
cypress/
├── e2e/              # Arquivos de teste
├── fixtures/         # Dados de teste
├── support/          # Commands customizados
└── README.md         # Esta documentação
```

## Como Executar

```bash
# Modo interativo
npx cypress open

# Modo headless
npx cypress run
```
